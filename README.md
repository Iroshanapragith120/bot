# 🤖 WhatsApp Memory Bot

මෙම බොට් මගින් ඔබගේ සුන්දර මතකයන් (Images) සහ පණිවිඩ (Messages) ඉතා පහසුවෙන් WhatsApp හරහා ඔබ කැමති වාර ගණනක් යැවීමට ඉඩ ලබා දෙයි. විශේෂයෙන්ම බිස්නස් වැඩ සහ මතකයන් බෙදා ගැනීමට මෙය නිර්මාණය කර ඇත.

### 🚀 විශේෂාංග (Features)

* **Image Support:** GitHub එකේ ඇති පින්තූරයම මැසේජ් එක ලෙස යැවීමේ හැකියාව.
* **Custom Caption:** `cha.js` මගින් ඕනෑම දිගු මැසේජ් එකක් යැවීමට හැකි වීම.
* **Interactive Terminal:** බොට් ක්‍රියාත්මක වන අතරතුර Number එක සහ Count එක ඇතුළත් කළ හැකිය.
* **Safety Delay:** WhatsApp අංකය ආරක්ෂා කර ගැනීමට තත්පර 3ක පරතරයක් (Delay) සහිතයි.
* **Stay Alive:** මැසේජ් යැවීමෙන් පසු බොට් අක්‍රිය නොවන අතර නැවත නැවතත් පාවිච්චි කළ හැකිය.

### 🛠️ පිහිටුවා ගන්නා ආකාරය (Installation)

1.  Repository එක Clone කරන්න:
    ```bash
    git clone [https://github.com/Iroshanapragith120/bot.git](https://github.com/Iroshanapragith120/bot.git)
    cd bot
    ```

2.  අවශ්‍ය Libraries Install කරන්න:
    ```bash
    npm install
    ```

3.  පින්තූරය එක් කරන්න:
    ඔබේ පින්තූරය `memory.jpg` නමින් මෙහි ප්‍රධාන ෆෝල්ඩරයට ඇතුළත් කරන්න.

4.  මැසේජ් එක වෙනස් කරන්න:
    `cha.js` ෆයිල් එක ඇතුළත ඔබට අවශ්‍ය Caption එක ලියන්න.
